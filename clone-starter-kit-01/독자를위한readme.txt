안녕하세요? 저는 이 책을 담당한 편집자P입니다.

01장에서 해야 할 일을 아래에 목록으로 정리했습니다.

- node.js LTS 버전 설치(www.nodejs.org 접속하여 설치 파일 다운로드)
- node.js, npm 설치 확인(node -v, npm -v 명령어로 확인)
- npm으로 npx 설치(npx는 최신 노드 패키지를 1회성으로 사용할 수 있게 해주는 노드 패키지입니다)
- 비주얼 스튜디오 코드 설치(code.visualstudio.com 접속하여 설치 파일 다운로드)
- git 설치(git-scm.com 접속하여 설치 파일 다운로드) 및 git 설치 확인(git --version 명령어로 확인)

이후 스타터키트를 사용하실 것인가요? 하지만 저는 직접 책을 따라 입력하면서 공부해 볼 것을 권장합니다.
이 책은 사용하도록 만들어졌기 때문이죠. 물론 코드 입력이 힘든 독자라면 스타터키트를 사용해도 좋습니다.
혹시 스타터키트를 사용하며 공부할 계획이 있으시다면 반드시!
